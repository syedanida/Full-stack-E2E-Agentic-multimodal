# Full Stack Multimodal Agent Assignment

This project demonstrates a **full-stack multimodal AI agent** integrating frontend, backend, and RAG (Retrieval-Augmented Generation) capabilities, along with a database backend for persistent storage.

### Full Stack Multimodal Personal Expense Assistant  
**Description:** Build a multimodal agent that can interact through text, images, and structured data. The system includes:  
- A **web frontend** for user interaction  
- An **ADK-based backend** for agent logic  
- **RAG (Retrieval-Augmented Generation)** for intelligent context retrieval  
- A **database** for persistent data storage and management
